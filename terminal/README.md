# Terminal Configuration

Terminal emulator configurations.

## WezTerm

GPU-accelerated terminal with Lua configuration.

### Installation
```bash
brew install --cask wezterm
ln -sf ~/dotfiles/terminal/wezterm/.wezterm.lua ~/.wezterm.lua
```

### Features

- Catppuccin Mocha theme
- MesloLGS Nerd Font
- 70% screen size on startup
- Transparency and blur effects

# Neovim Configuration

Modern Neovim setup with LSP, Treesitter, and quality-of-life plugins.

## Installation
```bash
brew install neovim ripgrep

# Symlink config
ln -sf ~/dotfiles/nvim ~/.config/nvim

# Open Neovim - plugins will auto-install
nvim
```

## Features

- LSP support (Python, Java, Nix, Bash, YAML, JSON, Markdown)
- Treesitter syntax highlighting
- Telescope fuzzy finder
- Auto-completion
- Git integration
- File explorer

# Shell Configuration

Zsh shell configuration with modern enhancements.

## Installation
```bash
# Install dependencies
brew install powerlevel10k zsh-autosuggestions zsh-syntax-highlighting zoxide eza

# Symlink config
ln -sf ~/dotfiles/shell/.zshrc ~/.zshrc

# Configure Powerlevel10k theme
p10k configure
```

## Features

- Powerlevel10k theme
- Autosuggestions and syntax highlighting
- Modern replacements: `eza` (ls), `zoxide` (cd)
- Smart history search
